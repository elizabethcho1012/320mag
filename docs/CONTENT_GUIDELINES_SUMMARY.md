# Third Twenty 콘텐츠 가이드라인 요약

## 🎯 핵심 원칙

### 1. 법적 안전성
- ❌ 원문 복붙 금지
- ❌ 단순 번역 금지
- ✅ 사실만 추출 + 완전한 재창작

### 2. 인간다움 (AI 티 방지)
- ❌ 로봇 같은 완벽한 문법
- ❌ "~했습니다" 뉴스 어투
- ✅ 친근한 말투, 개인적 경험, 감탄사

### 3. 독자 맞춤
- 타깃: 한국 시니어 (40-60대)
- 관점: 인생 경험, 실용성, 공감
- 길이: 2000-2500자

## 📝 작성 체크리스트

### 변형성 (Transformation)
- [ ] 원문과 구조 완전히 다름
- [ ] 시니어 관점으로 재해석
- [ ] 2000자 이상
- [ ] 출처 URL 포함

### 인간다움 (Human-like)
- [ ] 문장 길이 다양함
- [ ] 구어체 표현 포함
- [ ] 개인적 일화/경험
- [ ] 감탄사, 추임새
- [ ] 에디터 페르소나 뚜렷함

### 법적 안전성
- [ ] 원문 표현 직접 사용 안 함
- [ ] 번역기 느낌 없음
- [ ] 사실 정보만 활용
- [ ] 완전히 새로운 글

## ⚠️ 금지 표현

### AI 티 나는 표현
- "분석 결과"
- "데이터에 따르면"
- "객관적으로"
- "첫째, 둘째, 셋째"
- "결론적으로"

### 번역기 어투
- "~했습니다"
- "~것입니다"
- "~되었습니다"

## ✅ 권장 표현

### 친근한 시작
- "여러분, 안녕하세요"
- "요즘 제가 자주 듣는 질문이 있어요"
- "솔직히 말씀드리면"

### 개인적 경험
- "제가 지난주 만난 고객분은..."
- "30년 이 일을 하면서 배운 게..."
- "최근에 이런 일이 있었어요"

### 자연스러운 추임새
- "그런데 말이죠"
- "아, 그리고"
- "정말이에요"
- "맞아요"

## 📊 품질 기준

|------|------|------|
| 글자 수 | 2000자 | 2500자 |
| 문단 수 | 5개 | 7-10개 |
| 개인적 일화 | 1개 | 2-3개 |
| 실용 팁 | 3개 | 5개 |
| 구어체 표현 | 2번 | 5번 |
